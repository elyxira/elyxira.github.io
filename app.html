    远程仓库使用说明

    配置说明

    1. 创建GitHub仓库用于存储题库文件
    2. 生成GitHub Personal Access Token:
      • 访问GitHub Settings -> Developer settings -> Personal access tokens
      • 点击"Generate new token"
      • 勾选"repo"权限
      • 生成并保存token
    3. 在远程仓库配置中填写：
      • 仓库地址：你的GitHub用户名/仓库名
      • 分支名称：通常是main或master
      • 题库路径：题库文件存放的目录
      • GitHub Token：之前生成的token

    使用说明

    • 题库文件格式应为JSON格式
    • 启用自动同步后，笔记会自动保存到远程仓库
    • 也可以通过笔记编辑器中的同步按钮手动同步
    • 首次使用请先测试连接以确保配置正确

    注意事项

    • 请妥善保管GitHub Token，不要泄露给他人
    • 建议定期备份本地数据
    • 如遇到同步问题，请检查网络连接和Token权限
